# 📖 책 페이지 추출기 (브라우저 웹앱)

브라우저에서 바로 실행되는 웹앱입니다.
영상에서 책 페이지를 자동 추출하고, OCR로 텍스트를 인식한 후 Google Docs로 내보냅니다.

**설치 없이 브라우저에서 바로 사용 가능합니다.**

## 기능

- 영상에서 페이지 전환 자동 감지 (SSIM 기반, 중복 제거)
- Tesseract.js로 한국어/영어 텍스트 OCR 인식
- Google Docs에 자동 업로드
- 텍스트 파일(.txt) 다운로드 지원
- 드래그 앤 드롭 영상 업로드
- 실시간 진행 상황 표시

## 사용법

### 1. 웹앱 실행

`index.html`을 브라우저에서 열면 됩니다.

> ⚠️ 로컬 파일로 열 경우 일부 기능이 제한될 수 있으므로 로컬 서버를 추천합니다:
```bash
# Python
python -m http.server 8000

# Node.js
npx serve .
```
그 후 http://localhost:8000 접속

### 2. 영상 업로드 → 설정 조정 → "페이지 추출 시작" 클릭

### 3. 결과 확인 후
- **Google Docs에 업로드** 또는
- **텍스트 파일 다운로드**

## Google Docs 업로드 설정 (선택사항)

Google Docs 업로드를 사용하려면:

1. [Google Cloud Console](https://console.cloud.google.com) 접속
2. 프로젝트 생성 또는 선택
3. **Google Docs API** 활성화
4. **사용자 인증 정보** → **OAuth 2.0 클라이언트 ID** 생성
   - 애플리케이션 유형: **웹 애플리케이션**
   - 승인된 JavaScript 출처: `http://localhost:8000` (개발용)
5. `app.js` 상단의 `GOOGLE_CLIENT_ID`를 발급받은 ID로 교체

> Google Docs 없이도 텍스트 파일 다운로드는 바로 사용 가능합니다.

## 파일 구조

```

```

## 설정 가이드

| 설정 | 기본값 | 설명 |
|------|--------|------|
| 유사도 임계값 | 0.85 | 낮을수록 큰 변화만 새 페이지로 인식 |
| 프레임 간격 | 1초 | 분석 빈도. 낮을수록 정확하지만 느림 |
| OCR 언어 | 한국어+영어 | 인식할 텍스트 언어 |

## 팁

- 영상 해상도가 높을수록 OCR 정확도가 좋습니다
- 페이지 전환이 빠른 영상은 프레임 간격을 낮추세요 (0.3~0.5초)
- 같은 페이지가 반복 나오는 영상은 유사도 임계값을 낮추세요 (0.75~0.8)
